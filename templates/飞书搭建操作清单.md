# 飞书搭建操作清单

这份清单默认你从 0 开始新建一个飞书多维表格。

## 一、先建底表

1. 打开飞书。
2. 新建一个 `多维表格`。
3. 名字改成 `商家不断更排期表`。

## 二、导入商家总览表

1. 在左侧第一张表里，点表名改成 `商家总览表`。
2. 选择 `导入数据`。
3. 导入文件 [商家总览表_初始化.csv](/Users/douwenkai/Documents/codex/templates/商家总览表_初始化.csv)。
4. 导入后先检查 4 个商家是否都在。

### 商家总览表字段类型

按下面改字段类型：

- `商家名称`：单行文本
- `发片规则类型`：单选
- `发片周几`：多选
- `当前可发库存`：数字
- `每次拍摄产出`：数字
- `下一个发布日期`：日期
- `未来第1次发布日期`：日期
- `未来第2次发布日期`：日期
- `未来第3次发布日期`：日期
- `未来第4次发布日期`：日期
- `未来第5次发布日期`：日期
- `库存可支撑到`：公式
- `断更发布日期`：公式
- `断更前最后安全发布日期`：公式
- `最晚开始剪辑日`：公式
- `最晚拍摄日`：公式
- `最晚写脚本日`：公式
- `风险等级`：单选 或 公式
- `备注`：多行文本

### 商家总览表单选项

给 `发片规则类型` 加 3 个选项：

- `单数日`
- `双数日`
- `自定义周几`

给 `风险等级` 加 3 个选项：

- `红`
- `黄`
- `绿`

### 商家总览表公式列

先建最重要的 6 个公式列。

#### 1. 库存可支撑到

```text
IF(
  {当前可发库存}=1,{未来第1次发布日期},
  IF(
    {当前可发库存}=2,{未来第2次发布日期},
    IF(
      {当前可发库存}=3,{未来第3次发布日期},
      IF(
        {当前可发库存}=4,{未来第4次发布日期},
        IF(
          {当前可发库存}>=5,{未来第5次发布日期},
          BLANK()
        )
      )
    )
  )
)
```

#### 2. 断更发布日期

```text
IF(
  {当前可发库存}=0,{未来第1次发布日期},
  IF(
    {当前可发库存}=1,{未来第2次发布日期},
    IF(
      {当前可发库存}=2,{未来第3次发布日期},
      IF(
        {当前可发库存}=3,{未来第4次发布日期},
        IF(
          {当前可发库存}=4,{未来第5次发布日期},
          BLANK()
        )
      )
    )
  )
)
```

#### 3. 断更前最后安全发布日期

```text
{库存可支撑到}
```

#### 4. 最晚开始剪辑日

```text
DATEADD({断更发布日期},-1,"day")
```

#### 5. 最晚拍摄日

```text
DATEADD({断更发布日期},-2,"day")
```

#### 6. 最晚写脚本日

```text
DATEADD({断更发布日期},-3,"day")
```

如果你的飞书公式里 `DATEADD` 不可用，就先把这 3 列改成日期列，手动填；后面我再帮你改成你当前飞书版本能用的写法。

### 风险等级推荐先手动

第一版建议你先把 `风险等级` 当单选手动维护，最稳。

判断标准：

- 距离断更 `3天内`：`红`
- 距离断更 `4-7天`：`黄`
- 更久：`绿`

## 三、导入拍摄批次表

1. 新建第二张表。
2. 表名改成 `拍摄批次表`。
3. 选择 `导入数据`。
4. 导入文件 [拍摄批次表_初始化.csv](/Users/douwenkai/Documents/codex/templates/拍摄批次表_初始化.csv)。

### 拍摄批次表字段类型

- `批次名称`：单行文本
- `所属商家`：关联记录
- `计划拍摄条数`：数字
- `目标接上的发布日期`：日期
- `最晚写脚本日`：日期
- `最晚拍摄日`：日期
- `最晚开始剪辑日`：日期
- `脚本状态`：单选
- `拍摄状态`：单选
- `剪辑状态`：单选
- `当前阶段`：单选
- `是否逾期`：复选框 或 单选
- `备注`：多行文本

### 所属商家关联设置

1. 点 `所属商家` 字段。
2. 类型改成 `关联记录`。
3. 关联到 `商家总览表`。
4. 检查 `妙颜` 是否能正确匹配上。

### 拍摄批次表单选项

`脚本状态`：

- 未开始
- 进行中
- 已完成

`拍摄状态`：

- 未开始
- 已排期
- 已完成

`剪辑状态`：

- 未开始
- 进行中
- 已完成

`当前阶段`：

- 待写脚本
- 待拍摄
- 剪辑中
- 待交付
- 已完成
- 已逾期

## 四、开看板

1. 进入 `拍摄批次表`。
2. 新建视图。
3. 选择 `看板视图`。
4. 名字改成 `拍摄进度看板`。
5. 分组字段选择 `当前阶段`。

### 看板卡片上建议显示

把下面这些字段加到卡片里：

- `所属商家`
- `目标接上的发布日期`
- `最晚写脚本日`
- `最晚拍摄日`
- `最晚开始剪辑日`

## 五、建三个常用视图

### 1. 本周必须处理

1. 在 `拍摄批次表` 新建网格视图。
2. 名字改成 `本周必须处理`。
3. 筛选条件设置：
   - `目标接上的发布日期` 在本周内
   - 或 `最晚写脚本日` 在本周内

### 2. 48小时内要行动

1. 再建一个网格视图。
2. 名字改成 `48小时内要行动`。
3. 筛选条件设置：
   - `最晚写脚本日 <= 今天+2`
   - 或 `最晚拍摄日 <= 今天+2`
   - 或 `最晚开始剪辑日 <= 今天+2`

### 3. 已逾期

1. 再建一个网格视图。
2. 名字改成 `已逾期`。
3. 第一版先用简单条件：
   - `当前阶段 = 已逾期`

## 六、做颜色提醒

### 商家总览表

设置条件格式：

- `风险等级 = 红`：整行红色
- `风险等级 = 黄`：整行黄色
- `风险等级 = 绿`：默认

### 拍摄批次表

设置条件格式：

- `当前阶段 = 已逾期`：整行红色
- `当前阶段 = 已完成`：整行灰色

## 七、把你现在的真实情况填进去

先把这 4 家确认好：

- 妙颜：双数日更新
- 包记西点：单数日更新
- 西施竹韵：周一 / 周三 / 周五 / 周六 / 周日更新
- 川香居麻辣鸡块：单数日更新

然后你只做两件事：

1. 把每家的 `当前可发库存` 改成真实值。
2. 检查每家的 `未来第1次发布日期` 到 `未来第5次发布日期`。

## 八、先拿妙颜验收

妙颜这一行应该表现成这样：

- 当前可发库存：`3`
- 未来第1次发布日期：`2026-04-18`
- 未来第2次发布日期：`2026-04-20`
- 未来第3次发布日期：`2026-04-22`
- 未来第4次发布日期：`2026-04-24`
- 库存可支撑到：`2026-04-22`
- 断更发布日期：`2026-04-24`
- 最晚写脚本日：`2026-04-21`
- 最晚拍摄日：`2026-04-22`
- 最晚开始剪辑日：`2026-04-23`

如果这一行对了，整套表基本就跑通了。

## 九、你每天怎么用

1. 先看 `商家总览表` 的红黄风险。
2. 有风险的商家就去 `拍摄批次表` 建一条新批次。
3. 每推进一步，就改 `当前阶段`。
4. 一批交付后，把商家库存手动 `+3`。

配合 [每日使用SOP.md](/Users/douwenkai/Documents/codex/templates/每日使用SOP.md) 一起用会更顺。
